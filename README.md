A simple yet engaging Flappy Bird clone built in Java, featuring smooth gameplay, responsive controls, and classic obstacle mechanics. This project is perfect for beginners learning Java OOP, game loops, event handling, and 2D graphics.
Features:
Smooth Gameplay powered by Java graphics
Pipe Obstacles randomly generated
Custom Sprites for the bird and pipes
Sound Effects (optional if you add them later)
Score Tracking
Easy-to-understand Java code structure
Project Structure
/src
  App.java
  FlappyBird.java
  FlappyBird$Pipe.class
  FlappyBird$Bird.class
/assets
  flappybird.png
  bottompipe.png
  toppipe.png
  flappybirdbg.png
How to run the code :
javac *.java
java App
